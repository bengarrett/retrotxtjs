# RetrotxtJS

RetrotxtJS turns legacy ASCII and NFO text into in-browser HTML.

## Build

```bash
# Clone this repo.
git clone https://github.com/bengarrett/retrotxtjs
cd retrotxtjs

# Install dependencies.
yarn # or npm install
```
